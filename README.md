# 📊 Student Performance Data Analysis Dashboard

A web-based **Student Performance Data Analysis Dashboard** built with **Python, Flask, Pandas, NumPy, and Matplotlib**. The application analyzes student performance data from a CSV file and presents meaningful insights through interactive charts and statistics.

## 🌐 Live Demo

**Live Application:** https://data-analysis-project-ooy8.onrender.com

---

## 🚀 Features

* 📂 Load and analyze student performance data from CSV files.
* 📈 Calculate subject-wise average marks.
* 📊 Interactive data visualizations using Matplotlib.
* 📉 Scatter plot for study hours vs. average marks.
* 🔥 Correlation heatmap for numerical features.
* 📋 Display complete student dataset in a responsive dashboard.
* 🔄 REST API endpoints returning JSON data.
* 🌐 Fully deployed on Render.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/mbajiya047/data_analysis_project.git
```

Move into the project directory:

```bash
cd data_analysis_project
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
